# House Price Prediction & Market Analysis

## üè† Overview
This project demonstrates a complete **house price prediction pipeline** using machine learning (**XGBoost Regressor**) and interactive **Power BI dashboards** for data visualization and market insights.  
It is designed as a **portfolio-level project** to showcase data science and analytics skills, from data preprocessing to modeling and visualization.

---

## üìä Key Features
- **Data Cleaning & Preprocessing:** Handle missing values, encode categorical features, and prepare data for modeling.
- **Exploratory Data Analysis (EDA):** Identify trends, distributions, and correlations in housing data.
- **Feature Importance Analysis:** Determine which factors most influence house prices.
- **Predictive Modeling:** Train and evaluate an **XGBoost Regressor**:
  - **RMSE:** 0.06  
  - **MAE:** 0.04  
  - **R¬≤ Score:** 0.977
- **Power BI Dashboard:** Interactive visualizations including:
  - House price trends by year built  
  - Price distribution by neighborhood and overall quality  
  - Scatter plots to analyze price vs. living area  
  - Slicers for dynamic filtering

---

## üìÇ Dataset
- **Source:** [Kaggle: House Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)  
- **Training samples:** 1460  
- **Features:** 81 (numerical & categorical)  
- **Target:** SalePrice
